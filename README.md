## Table of Contents


## Install

[npm][]:

```sh
npm install iscloudy
```

[yarn][]:

```sh
yarn add iscloudy
```


## Usage

```js
const Iscloudy = require('iscloudy');

const iscloudy = new Iscloudy();

console.log(iscloudy.check());
// script
```
